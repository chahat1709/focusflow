========================================================================
FOCUS FLOW - Quick Start Guide
========================================================================

🚀 HOW TO START
------------------------------------------------------------------------
1. Put FocusFlow.exe and config.json in a folder
2. Double-click FocusFlow.exe
3. Wait for your browser to open
4. Click "Connect Muse" button (top-right)
5. Done!

📋 WHAT YOU NEED
------------------------------------------------------------------------
✓ Windows 10 or 11
✓ Muse headset (turned ON)
✓ Bluetooth enabled

📁 YOUR DATA
------------------------------------------------------------------------
All recordings saved in: recordings/ folder

📖 NEED HELP?
------------------------------------------------------------------------
Read USER_MANUAL.txt for detailed instructions
Check focus_flow.log if something goes wrong
